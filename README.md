# CLI Scholar RPG 🎲🏫

Um jogo de suspense e terror baseado em texto desenvolvido para ser executado diretamente no terminal (CLI - Command Line Interface). Este projeto foi criado como um **trabalho escolar**, unindo o desenvolvimento de lógica de programação com uma narrativa interativa e imersiva, onde as escolhas do jogador determinam o seu destino e a sua sobrevivência.

---

## 🚀 Funcionalidades

- **Narrativa Interativa:** Uma história envolvente de suspense e terror onde cada escolha conta.
- **Interface CLI:** Toda a experiência ocorre diretamente no terminal, utilizando o poder do texto para construir a atmosfera de medo.
- **Tomada de Decisões:** Menus interativos para o jogador escolher o seu próximo passo em tempo real.
- **Múltiplos Finais:** Suas decisões determinam se você sobreviverá ou se tornará mais uma vítima.
